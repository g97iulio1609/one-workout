# Exercise Selection Skill

## Compound vs Isolation Balance

| Goal        | Compound % | Isolation % |
| ----------- | ---------- | ----------- |
| Strength    | 70%        | 30%         |
| Hypertrophy | 55%        | 45%         |
| Endurance   | 60%        | 40%         |
| Power       | 80%        | 20%         |

## Exercise Selection Priority by Goal

### Strength Focus

1. **Big 3 First**: Squat, Bench Press, Deadlift
2. **Competition Variations**: Close-grip bench, pause squat
3. **Compound Accessories**: Romanian deadlift, overhead press
4. **Targeted Isolation**: Triceps, hamstrings

### Hypertrophy Focus

1. **Volume Compounds**: Higher rep bench, incline, rows
2. **Multiple Angles**: Incline, flat, decline for chest
3. **Isolation Variety**: Different curl variations, lateral raises
4. **Mind-Muscle Connection**: Cable work, machines

## Muscle Group Minimums

Each target muscle group needs at least:

- 2 distinct exercises
- 1 compound movement (when applicable)
- 10+ weekly sets total
