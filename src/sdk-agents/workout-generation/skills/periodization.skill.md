# Periodization Skill

## Training Phase Characteristics

### Accumulation Phase (2-3 weeks)

- **Volume**: High (100-110% baseline)
- **Intensity**: Moderate (70-80% 1RM)
- **RPE Range**: 6-8
- **Rep Range**: 8-15
- **Rest Periods**: 60-120 seconds
- **Focus**: Building work capacity, technique refinement

### Intensification Phase (2-3 weeks)

- **Volume**: Moderate (80-90% baseline)
- **Intensity**: High (80-90% 1RM)
- **RPE Range**: 8-9
- **Rep Range**: 3-8
- **Rest Periods**: 120-180 seconds
- **Focus**: Strength expression, neural adaptations

### Realization Phase (1-2 weeks)

- **Volume**: Low (60-70% baseline)
- **Intensity**: Peak (90-100% 1RM)
- **RPE Range**: 9-10
- **Rep Range**: 1-5
- **Rest Periods**: 180-300 seconds
- **Focus**: Peak performance, testing

### Deload Phase (1 week)

- **Volume**: Very Low (40-60% baseline)
- **Intensity**: Low (50-70% 1RM)
- **RPE Range**: 5-6
- **Rep Range**: 8-12
- **Rest Periods**: 60-90 seconds
- **Focus**: Recovery, fatigue dissipation

## Deload Frequency

| Experience Level | Deload Frequency |
| ---------------- | ---------------- |
| Beginner         | Every 4-6 weeks  |
| Intermediate     | Every 3-4 weeks  |
| Advanced         | Every 2-3 weeks  |

## Block Structure Examples

### 8-Week Strength Block

```
Weeks 1-3: Accumulation
Week 4: Deload
Weeks 5-7: Intensification
Week 8: Realization
```

### 4-Week Hypertrophy Block

```
Week 1-3: Accumulation (progressive volume)
Week 4: Deload
```
